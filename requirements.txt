## Installation

### Quick Install (Basic)
```bash
pip install mutagen librosa numpy scipy opencv-python pillow psutil
```

### Full Install (Recommended)
```bash
pip install -r requirements.txt
```

### GPU Support (CUDA)
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### GPU Support (Apple Silicon)
```bash
pip install --pre torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/nightly/cpu
```

## System Requirements

### Minimum
- 4GB RAM
- 2-core CPU
- 10GB free storage

### Recommended
- 16GB+ RAM
- 8+ core CPU
- NVIDIA/AMD GPU with 4GB+ VRAM
- 50GB free storage

### Optimal
- 32GB+ RAM
- 16+ core CPU
- NVIDIA RTX 3060+ or equivalent
- 200GB+ free storage

## Optional Hardware Acceleration

### CUDA (NVIDIA GPUs)
- CUDA Toolkit 11.8+
- cuDNN 8.0+

### Metal (Apple Silicon)
- macOS 12.0+
- Automatic support via PyTorch

### OpenCL (AMD/Intel GPUs)
- OpenCL SDK installed
- PyTorch OpenCL backend
